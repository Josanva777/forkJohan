Encabezado y Barra de Navegación

El encabezado de la página incluye:

  • Logos: Dos logos que redirigen a la página de disponibilidad.
  • Botón de Menú: Un botón para mostrar y ocultar el menú de navegación lateral.
  • Logo de Cierre de Sesión: Un logo que redirige a la página de inicio de sesión.

La barra de navegación lateral, visible al hacer clic en el botón de menú, permite acceder a las siguientes secciones:

  • Disponibilidad
  • Facturas (sección activa)
  • Mensualidad
  • Reservas
  • Reportes
  • Registro
  • Ayuda

Sección de Disponibilidad

Esta sección permite al personal del parqueadero ver el estado de las celdas disponibles y ocupadas tanto para motos como para carros. Se presentan gráficas que ayudan a visualizar la disponibilidad de manera clara y rápida.

Motos

Para las motos, se muestran las siguientes gráficas:

  • Celdas Generales: Muestra el estado general de todas las celdas disponibles para motos.
  • Celdas de Reservas: Muestra el estado de las celdas reservadas para motos.
  • Celdas de Mensualidad: Muestra el estado de las celdas ocupadas por clientes con mensualidad para motos.

Carros

Para los carros, se muestran las siguientes gráficas:

  • Celdas Generales: Muestra el estado general de todas las celdas disponibles para carros.
  • Celdas de Reservas: Muestra el estado de las celdas reservadas para carros.
  • Celdas de Mensualidad: Muestra el estado de las celdas ocupadas por clientes con mensualidad para carros.

Este sistema de disponibilidad está diseñado para facilitar la gestión del parqueadero, permitiendo al personal ver rápidamente el estado de las celdas y optimizar el uso del espacio disponible tanto para motos como para carros.

Sección de Facturas

Esta sección permite al personal del parqueadero gestionar y registrar las facturas de los clientes. Aquí están las funcionalidades principales:

Formulario de Ingreso de Datos

  • Placa: Un campo para ingresar la placa del vehículo.
  • Hora de Entrada y Salida: Campos para registrar la hora de entrada y salida del vehículo.
  • Tipo de Vehículo: Opciones para seleccionar si es una moto o un carro.
  • Botones de Acción:
    • Imprimir: Para imprimir los datos ingresados.
    • Guardar: Para guardar la información ingresada.

Tabla de Registros

Incluye las siguientes columnas para cada registro:

  • Placa
  • Tarifa (Moto o Carro)
  • Hora de Entrada
  • Hora de Salida
  • Total de Tiempo
  • Recibo (enlace para ver detalles)
  • Eliminar (botón para eliminar el registro)

Esta tabla muestra todos los registros de facturación ingresados, permitiendo al personal revisar y gestionar las facturas de manera eficiente.

Sección de Mensualidad

Esta sección permite gestionar los registros de mensualidad de los clientes, ingresando y visualizando los datos necesarios para el control de las tarifas y tiempos de uso de los espacios del parqueadero.

Formulario de Ingreso de Datos

El formulario incluye los siguientes campos:

  • Placa: Un campo de texto para ingresar la placa del vehículo.
  • Día de Entrada y Salida: Dos campos de fecha para registrar el día de entrada y salida del vehículo.
  • Tarifa: Opciones para seleccionar si el vehículo es una moto o un carro.

El formulario también incluye botones para:

  • Imprimir: Para generar un recibo impreso.
  • Guardar: Para guardar los datos ingresados.

Tabla de Registros

La tabla muestra los registros de los vehículos con mensualidad, incluyendo la siguiente información:

  • Placa: La placa del vehículo.
  • Tarifa: Si el vehículo es una moto o un carro.
  • Hora de Entrada y Salida: Las horas de entrada y salida del vehículo.
  • Total Tiempo: El tiempo total que el vehículo ha permanecido en el parqueadero.
  • Recibo: Un enlace para ver el recibo del vehículo.
  • Eliminar: Una opción para eliminar el registro del vehículo.

El sistema de mensualidad está diseñado para facilitar la gestión de los clientes con mensualidad, permitiendo registrar y visualizar fácilmente los datos relevantes,optimizando así la usuarioistración del parqueadero.

Sección de Reservas

Esta sección permite gestionar las reservas de espacios en el parqueadero, ingresando y visualizando los datos necesarios para organizar las reservas de los clientes.

Formulario de Ingreso de Datos

El formulario incluye los siguientes campos:

  • Placa: Un campo de texto para ingresar la placa del vehículo.
  • Día de Reserva: Un campo de fecha para registrar el día de reserva del espacio.
  • Hora de Entrada y Salida: Dos campos de hora para registrar la hora de entrada y salida del vehículo.
  • Tarifa: Opciones para seleccionar si el vehículo es una moto o un carro.

El formulario también incluye botones para:

  • Imprimir: Para generar un recibo impreso.
  • Guardar: Para guardar los datos ingresados.

Tabla de Reservas

La tabla muestra las reservas de espacios en el parqueadero, incluyendo la siguiente información:

  • Placa: La placa del vehículo reservado.
  • Tarifa: Si el vehículo reservado es una moto o un carro.
  • Hora de Entrada y Salida: Las horas de entrada y salida del vehículo reservado.
  • Total Tiempo: El tiempo total de la reserva.
  • Recibo: Un enlace para ver el recibo de la reserva.
  • Eliminar: Una opción para eliminar la reserva.

El sistema de reservas está diseñado para facilitar la gestión de las reservas de espacios en el parqueadero, permitiendo registrar y visualizar fácilmente los datos relevantes, optimizando así la organización de los espacios disponibles.

Sección de Reportes

Esta sección permite generar reportes de ingresos del parqueadero, ofreciendo opciones para filtrar los datos según diferentes criterios.

Gráfico de Ingresos

El gráfico muestra los ingresos del parqueadero en un período de tiempo determinado.

Filtros de Reporte

Los filtros permiten seleccionar el tipo de vehículo y el período de tiempo del reporte:

  • Tipo de Vehículo: Opciones para seleccionar si el reporte es para carros o motos.
  • Período de Tiempo: Botones para seleccionar el período del reporte (diario, semanal, mensual o anual).

La página de reportes proporciona una forma fácil y visual de analizar los ingresos del parqueadero, permitiendo a los usuarioistradores tomar decisiones informadas basadas en los datos recopilados.

Sección de Registro de Usuarios

Esta sección permite registrar nuevos usuarios en el sistema del parqueadero.

Formulario de Registro

El formulario recopila la siguiente información del usuario:

  • Nombre y Apellidos
  • Tipo de Documento y Número de Documento
  • Sexo
  • Teléfono
  • Correo Electrónico
  • Rol y Cargo
  • Usuario y Contraseña

El formulario también incluye botones para eliminar o guardar el registro.

La página de registro proporciona una interfaz para agregar nuevos usuarios al sistema del parqueadero, permitiendo usuarioistrar sus roles y permisos de acceso.
