Encabezado y Barra de Navegación

El encabezado de la página incluye:

  • Logos: Dos logos que redirigen a la página de disponibilidad.
  • Botón de Menú: Un botón para mostrar y ocultar el menú de navegación lateral.
  • Logo de Cierre de Sesión: Un logo que redirige a la página de inicio de sesión.

La barra de navegación lateral, visible al hacer clic en el botón de menú, permite acceder a las siguientes secciones:

  • Disponibilidad
  • Facturas (sección activa)
  • Mensualidad
  • Reservas
  • Reportes
  • Registro
  • Ayuda

Sección de Disponibilidad

Esta sección permite al personal del parqueadero ver el estado de las celdas disponibles y ocupadas tanto para motos como para carros. Se presentan gráficas que ayudan a visualizar la disponibilidad de manera clara y rápida.

Motos

Para las motos, se muestran las siguientes gráficas:

  • Celdas Generales: Muestra el estado general de todas las celdas disponibles para motos.
  • Celdas de Reservas: Muestra el estado de las celdas reservadas para motos.
  • Celdas de Mensualidad: Muestra el estado de las celdas ocupadas por clientes con mensualidad para motos.

Carros

Para los carros, se muestran las siguientes gráficas:

  • Celdas Generales: Muestra el estado general de todas las celdas disponibles para carros.
  • Celdas de Reservas: Muestra el estado de las celdas reservadas para carros.
  • Celdas de Mensualidad: Muestra el estado de las celdas ocupadas por clientes con mensualidad para carros.

Este sistema de disponibilidad está diseñado para facilitar la gestión del parqueadero, permitiendo al personal ver rápidamente el estado de las celdas y optimizar el uso del espacio disponible tanto para motos como para carros.

Sección de Facturas

Esta sección permite al personal del parqueadero gestionar y registrar las facturas de los clientes. Aquí están las funcionalidades principales:

Formulario de Ingreso de Datos

  • Placa: Un campo para ingresar la placa del vehículo.
  • Hora de Entrada y Salida: Campos para registrar la hora de entrada y salida del vehículo.
  • Tipo de Vehículo: Opciones para seleccionar si es moto o carro.
  • Generar Factura: Botón para calcular y generar la factura correspondiente.

Historial de Facturas

Una tabla que muestra el historial de facturas generadas, con información como:

  • Placa del vehículo
  • Hora de entrada y salida
  • Tipo de vehículo
  • Monto total

Sección de Mensualidad

Esta sección está diseñada para gestionar las mensualidades de los clientes que utilizan el parqueadero de forma regular. Aquí se encuentran las principales funcionalidades:

Registro de Cliente

  • Datos Personales: Campos para ingresar nombre, número de identificación, y otros datos relevantes.
  • Información del Vehículo: Campos para registrar la placa y tipo de vehículo.
  • Periodo de Mensualidad: Selección del periodo de tiempo para la mensualidad.
  • Botón de Registro: Botón para registrar al cliente en el sistema de mensualidades.

Listado de Clientes con Mensualidad

Una tabla que muestra los clientes actuales con mensualidad, incluyendo:

  • Nombre del cliente
  • Número de identificación
  • Placa del vehículo
  • Periodo de mensualidad
  • Estado del pago

Este sistema facilita el seguimiento y la gestión de los clientes con mensualidad, asegurando que el personal del parqueadero pueda verificar el estado de los pagos y la información del cliente de manera eficiente.

Sección de Reservas

En esta sección, el personal del parqueadero puede gestionar las reservas de celdas para los clientes. Las funcionalidades incluyen:

Formulario de Reserva

  • Datos del Cliente: Campos para ingresar nombre y número de identificación del cliente.
  • Información del Vehículo: Campos para registrar la placa y tipo de vehículo.
  • Fecha y Hora de Reserva: Selección de la fecha y hora en la que se realizará la reserva.
  • Botón de Confirmar Reserva: Botón para registrar la reserva en el sistema.

Listado de Reservas

Una tabla que muestra las reservas realizadas, incluyendo:

  • Nombre del cliente
  • Número de identificación
  • Placa del vehículo
  • Fecha y hora de la reserva
  • Estado de la reserva

Sección de Reportes

Esta sección permite al personal del parqueadero generar y revisar reportes sobre las actividades y estados del parqueadero. Las principales funcionalidades incluyen:

Generación de Reportes

  • Seleccionar Tipo de Reporte: Opciones para seleccionar el tipo de reporte (ej. disponibilidad, mensualidades, facturas).
  • Rango de Fechas: Campos para ingresar el rango de fechas para el reporte.
  • Botón de Generar Reporte: Botón para generar el reporte en base a los parámetros seleccionados.

Visualización de Reportes

Una tabla o gráfica que muestra la información del reporte generado, permitiendo al personal analizar los datos y tomar decisiones informadas.

Sección de Registro

En esta sección, el personal del parqueadero puede registrar nuevos usuarios y gestionar sus datos. Las funcionalidades incluyen:

Registro de Usuario

  • Datos Personales: Campos para ingresar nombre, número de identificación, y otros datos relevantes.
  • Información del Vehículo: Campos para registrar la placa y tipo de vehículo.
  • Credenciales de Acceso: Campos para crear un nombre de usuario y contraseña.
  • Botón de Registro: Botón para registrar al usuario en el sistema.

Gestión de Usuarios

Una tabla que muestra los usuarios registrados, incluyendo:

  • Nombre del usuario
  • Número de identificación
  • Placa del vehículo
  • Tipo de usuario
  • Estado del usuario

Este sistema permite una gestión eficiente de los usuarios del parqueadero, facilitando el registro y la administración de sus datos y credenciales de acceso.

Documentación detallada sobre cómo utilizar cada sección del sistema, con instrucciones paso a paso.
